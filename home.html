Codiners가 무엇인가요?

Coniders는 당신만의 코드를 언제든지 저장하고 다시 열람할 수 있는 공간입니다. 저희는 더 편한 세상을 위해 고생하시는 모든 개발자분들에게 더 좋은 개발환경을 제공해 드리기 위해 모든 서비스를 무료로 제공하고 있습니다.

코드 업로드 방법은?

1. 먼저 로그인을 합니다.
2. 등록된 코드가 없다면 화면 중앙에 위치한 +버튼을 눌러주세요.
3. 이미 코드가 등록되어 있다면, 코드 리스트의 첫번째 자리의 +버튼을 눌러주세요.
4. 코드 입력 페이지에서 코드 제목, 코드 내용, 코멘트를 입력후 등록 버튼을 눌러주세요.
